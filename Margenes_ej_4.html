El html

HTML (Lenguaje de Marcado de Hipertexto) es el código que se utiliza para estructurar y mostrar una página web y su contenido. Es el componente fundamental de la web y define la estructura de un documento mediante etiquetas

-El margin top, establece un margen superior de 200 px

-El margin right, establece un margen desde la derecha de 150 px

-El margin bottom, establece un inferior de 200 px

-El margin left, establece un espacio desde la izquierda de 150 px
